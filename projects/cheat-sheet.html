HTML and CSS Reference Sheet

➣ Standard HTML Document Structure

<!DOCTYPE html>

<html lang="en">


<head>

 <meta charset="utf-8">

 <link rel="stylesheet" href="Location & Name of CSS Style Sheet">

 <title> Title of the document </title>

</head>


<body>

 The content of the document......

</body>


</html>

← This must be the very first thing in your HTML document

← This tag Defines an HTML document


← This tag defines the container for all the head elements

← This tag specifies the character encoding for the document

← This tag links to an external CSS style sheet

← This tag is a container for the title of the HTML document

← The head container can also include scripts, styles, meta, & more


← This tag defines the container for body of the HTML document

   The body contains all the content on the HTML document

   Such as: text, hyperlinks, images, tables, lists, etc.


← This tag Defines the end of the HTML document

➣ Commonly Used HTML Tags

Tag Usage
<div> </div> Create a new Division
<section> </section> Create a new Section
<p> </p> Create a new Paragraph
<h1> ➜ <h6> HTML Heading Sizes from the largest 1 to the smallest 6
<b> </b> Creates Bold Text
<u> </u> Creates Underlined Text
<i> </i> Creates Italic Text
<tt> </tt> Creates Teletype or Typewritter Style Text
<br> Inserts a Line Break
<hr> Inserts a Horizontal Line
<blockquote> </blockquote> Indents the text from both sides
<ol> </ol> Creates an Ordered List
<ul> </ul> Creates an Unordered List
<li> </li> Precedes each list item, adds a number or symbol depending upon the type of list selected
<table> </table> Creates a Table (Just Like This One)
<th> </th> Sets off the table header (a normal cell with bold, centered text)
<tr> </td> Sets off each row in the table
<td> </td> Sets off cell in a row
<img src="file/url"> Places an image in the HTML document
<a href="file/url">description</a> Places an external link in the HTML document

➣ Commonly Used CSS Property Groups

Properties Usage
.class-name{ } Class selector: elements that are styled separately but reoccur
#id-name{ } ID selector: styles unique elements
background-color: green; Color entire background of page
font-family: arial; Sets font
padding: 2em Sets width of negative space between content and border
display: inline-block; Blocks can align next to eachother but their size will not be affected.
float: right; 'Floats' elment to the right side of the page unless obstucted by another element.
