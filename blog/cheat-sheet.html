HTML and CSS Reference Sheet

Tag Usage
<!DOCTYPE> Defines the document type
<html> Defines an HTML document
<title> Defines a title for the document
<body> Defines an HTML document
<h1> to <h1> HTML Heading Sizes from 1 to 6
<p> Defines an HTML document
<header> Defines a header for a document or section
<footer> Defines a footer for a document or section
<div> Defines a title for the document
<section> Defines a section in a document
<article> Defines a article in a document
Tag Usage
#class-name{ } Class selector: elements that are styled separately but reoccur
.id-name{ } ID selector: styles unique elements
background-color: green; Color entire background of page
font-family: arial; Sets font
padding: 2em Sets width of negative space between content and border
display: inline-block; Blocks can align next to eachother but their size will not be affected.
float: right; 'Floats' elment to the right side of the page unless obstucted by another element.
